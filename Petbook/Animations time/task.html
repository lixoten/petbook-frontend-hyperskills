Description

It's time to add some interactive animations to the Petbook!

You can use CSS transitions to make your site more entertaining. Let's add these transitions to the items in the navbar when the mouse goes over them:

Use transform to scale or rotate the elements.
  • Make the logo image "Petbook" increase its size.
  • Make the pet-profile link image rotate along the x-axis.
  • Make the friends link image rotate along the y-axis.
  • Make the settings link image rotate along the z-axis.

For all of the above transitions, their durations should be 500 milliseconds. If you'd like, you can add more transitions to different elements.

Don't move your mouse when checking the animations as it can create an issue for the tests!
