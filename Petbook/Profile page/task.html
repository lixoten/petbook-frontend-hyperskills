Description

Make a good-looking profile page for your pet! The profile.html will contain a profile card and a description. Also, add information to the navbar and the footer.

The images in the navbar should also be anchor links and pet-profile-link should direct to the current HTML file. These links shouldn't have any text-decoration. All images should have an appropriate alt attribute in case the image can't be displayed.

Pay attention to the CSS rules and the elements in the Figma designs. The element names are the ids of your HTML elements. Also, notice the element structure in Figma; it hints at your HTML element structure.

You don't need to follow the styles for the following: font-family, font-size, font-weight and line-height .

Different from the Figma designs keep all of the images in their original widths!

Note the assets folder inside the src directory. It contains the images you need for the project.

The tests will only check the first three pieces of information in the description section: Name, animal, and breed. You can choose to add the rest. Also, from the socials in the footer, the tests will only check for the first three: Facebook, Twitter, and Instagram.

Objectives

Please, take a look at the designs made in Figma. In this stage, you must build only the design from Stage 2: Profile page.
