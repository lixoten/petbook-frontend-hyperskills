Description

Make the website elements display correctly on devices with different screen sizes.

In the Figma designs, you have three different screen widths: desktop(1200), tablet(768), and mobile(425). All previous stages were done according to the desktop size, so no change is needed.

If you've been using flex-box, changing the direction for different screen sizes should be easy.

Don't forget to review the page and change the device width from the device toolbar.
Objectives

Please, take a look at the designs made in Figma. This time you need to build only the design from Stage 6: Making it responsive.
